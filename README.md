# USTC 数字电路实验 大作业

该项目使用 python 生成电路文件，实现 LED 阵列滚动显示字符串。

python 版本： ```3.7.x```

PyGame 版本： ```2.1.0```

Logisim 版本： ```2.7.1```

```bash
$ python Start.py
```

---

目前已实现的功能：

 - 生成所有 ```Wire``` 、 ```Rom``` 、 ```DotMatrix``` 、 ```clock```

 - 根据字符串长度自动生成对应数量的原件，并调整位宽

 - 滚动显示整个字符串
